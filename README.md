# Digital Parking Permit System

##  Project Description
This project is a Digital Parking Permit System designed for a university. It allows students and staff to register their vehicles, request parking permits, check available parking zones, and report issues. Security staff can verify permits and manage violations.



##  Tools Used
- Google Docs (final report)
- Google Sheets (test cases)
- diagrams.net (workflow diagram)
- GitHub (version control)



##  System Features
- Vehicle Registration
- Parking Permit Request
- Parking Zone Availability Check
- Permit Approval Process
- Parking Verification by Security Staff


##  Workflow
Vehicle Registration → Permit Request → Zone Check → Permit Approval → Parking Verification

##  Defect Analysis
A defect was identified where a valid parking permit was incorrectly shown as expired during verification. This issue is analyzed in detail in the final report, including possible causes and regression testing.
